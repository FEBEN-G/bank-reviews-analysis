# Customer Experience Analytics for Fintech Apps

## Complete Project Implementation

This project analyzes Google Play Store reviews for three Ethiopian banks:
1. Commercial Bank of Ethiopia (CBE)
2. Bank of Abyssinia (BOA)
3. Dashen Bank

## Tasks Completed:
 **Task 1**: Data Collection & Preprocessing (1200+ reviews)
 **Task 2**: Sentiment & Thematic Analysis (100% coverage)
 **Task 3**: PostgreSQL Database Implementation (1200+ records)

## How to Run:
```bash
pip install -r requirements.txt
python scripts/task1_main.py
python scripts/task2_main.py
python scripts/task3_main.py
